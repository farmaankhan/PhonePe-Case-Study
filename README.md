# PhonePe-Case-Study

# Description
Analyze PhonePe transaction and demographic data across Indian states and districts to uncover trends, device usage patterns, and demographic correlations through advanced data analysis and consistency checks

# Technologies Used
python , Numpy , Pandas , Matplot

# Usage
1.Identified regional transaction trends to highlight high and low digital payment adoption areas.
2.Correlated demographic factors with transaction data to assess financial inclusion and user behavior.
3.Analyzed device usage patterns to derive insights for app optimization and targeted marketing.
